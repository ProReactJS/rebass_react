Getting started

Install

npm install --save rebass

Require

var Rebass = require('rebass');
var Button = Rebass.Button;
var Badge = Rebass.Badge;

Use

<App>
  <Button>Button</Button>
  <Badge>Badge</Badge>
</App>

Requires Basscss

Download Basscss
