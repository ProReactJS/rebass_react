Getting Started

Install

npm install rebass

Require

var Rebass = require('rebass');
var Button = Rebass.Button;
var Badge = Rebass.Badge;

Use

<App>
  <Button>Button</Button>
  <Badge>Badge</Badge>
</App>

Note: Rebass components require Basscss for styling.
