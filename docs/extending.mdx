
# Extending

Rebass components are great to use as base components that can be extended for various purposes in your design system.

## Container

```jsx
const Container = props =>
  <Box
    {...props}
    mx='auto'
    css={{
      maxWidth: '1024px'
    }}
  />
```

## Badge

```jsx
const Badge = props =>
  <Card
    color='white'
    bg='blue'
    {...props}
    px={3}
    py={1}
    borderRadius={9999}
    css={{
      display: 'inline-block'
    }}
  />
```

```.jsx
<Card
  color='white'
  bg='blue'
  px={3}
  py={1}
  borderRadius={9999}
  fontSize={0}
  css={{
    display: 'inline-block',
    textTransform: 'uppercase',
    letterSpacing: '.2em'
  }}>
  Badge
</Card>
```

## Avatar

```jsx
const Avatar = props =>

```

## BlockLink

```jsx
const BlockLink = props =>
  <Link
    color='inherit'
    {...props}
    css={{
      display: 'block',

    }}
  />
```

## NavLink

```jsx
const NavLink = props =>
  <Link
    px={2}
    py={1}

    css={{

      fontWeight: 'bold',
    }}
  />
```

## Embed

```jsx
const Embed = props =>
  <Box
    {...props}
    width={1}
    css={{
      height: 0,
      paddingBottom: (9 / 16) + '%',
      position: 'relative',
      overflow: 'hidden',
      '& > iframe': {
        position: 'absolute',
        width: '100%',
        height: '100%',
        top: 0,
        bottom: 0,
        left: 0,
        border: 0
      }
    }}
  />
```
