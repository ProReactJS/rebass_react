# Rebass

React Stateless Function UI Components Built with Basscss

http://jxnblk.com/rebass

## Getting Started

### Install

```bash
npm i rebass
```

### Import CSS

### Import Components

```js
import { Button, Badge } from 'rebass'
```

### Use

```xml
<App>
  <Button>Button</Button>
  <Badge>Badge</Badge>
</App>
```
