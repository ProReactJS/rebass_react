# Rebass

React UI components for Basscss (WIP)

http://jxnblk.com/rebass

## Getting Started

### Install

```bash
npm install --save rebass
```

### Require

```javascript
var Rebass = require('rebass');
var Button = Rebass.Button;
var Badge = Rebass.Badge;
```

### Use

```jsx
<App>
  <Button>Button</Button>
  <Badge>Badge</Badge>
</App>
```
